If for any reason you don't want to use the simpler and quicker One Click Installation method and want to make a Manual Installation, you will need to:

  1. download the latest Vanilla Forum core
  2. extract the file to a desired place
    vanilla-core-2-1-3.zip → DESKTOP\
  3. now you should have something like:
    DESKTOP\vanilla\
  4. extract the One Click theme that's on this pack to the forum themes directory
    ONE_CLICK_PACK\one-click-installation\themes\oneClick.zip → DESKTOP\vanilla\themes\
  5. now you should have something like:
    DESKTOP\vanilla\themes\oneClick\
  6. copy the whole vanilla folder to the public_html directory on your server through FTP
    DESKTOP\vanilla → SERVER_ROOT/public_html/
  7. open a browser and go to
    http://yourdomain.com/vanilla/
  8. fill the configurations form and press CONTINUE
    • if asked, you may need to create a DATABASE
      • go to your PhpMyAdmin and run this sql query:
        CREATE DATABASE vanilla CHARACTER SET utf8 COLLATE utf8_general_ci;
      • or create the DB in your cPanel or something similar
  9. you will be automatically redirected to the Vanilla Dashboard
  10. go to VANILLA > APPERANCE > THEMES
    • Apply the One Click - Premium Vanilla Theme
  11. have fun!

This won't install any external plugin.
For more information on how to add new plugins to your forum, you can read the Plugins page.

You can now choose which style you want to use, so please go to Customization.
