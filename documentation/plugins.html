One of the great features of Vanilla Forums is the ability to use plugins that adds new features to your forum.

The One Click Installation downloads and activates automatically the most commom plugins so that you don't loose too many time configuring them. But, if you decide that you don't want to use one of the plugins installed or you decide that you want to try one of the other available plugins, you can manage them by going to:

VANILLA BACKOFFICE > ADDONS > PLUGINS
If you want to install new plugins, you need to:
  1. download the desired plugin
  2. extract the downloaded file: plugin.zip → DESKTOP\
  3. copy the plugin folder to the plugins directory on your server through FTP DESKTOP\plugin_name → SERVER_ROOT/public_html/forum/plugins/
  4. go to VANILLA BACKOFFICE > ADDONS > PLUGINS
  5. Enable the plugin
