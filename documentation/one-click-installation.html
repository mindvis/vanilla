One Click comes with an automatic installation that will do all the hard work for you.
This is the recommended installation method but if you want you can make a Manual Installation.

So, in order to use the One Click Installation you will need to:

  1. copy the one-click-installation directory to your SERVER in the public_html directory
  2. now you should have something like:
    SERVER_ROOT/public_html/one-click-installation/
  3. open a web browser and go to:
    http://YOURDOMAIN.COM/one-click-installation/
  4. fill the form
  5. click on the ONE CLICK INSTALLATION button
    • if asked, you may need to create a DATABASE
      • go to your PhpMyAdmin and run this sql query:
        CREATE DATABASE vanilla CHARACTER SET utf8 COLLATE utf8_general_ci;
      • or create the DB in your cPanel or something similar
  6. after the installation is complete, click on the button inside the success message to access your new forum
  7. That's it! Simple, right?

You can now choose which style you want to use, so please go to Customization.
